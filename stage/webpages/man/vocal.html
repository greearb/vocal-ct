Name

VOCAL -- the Vovida Open Communication and Application Library

Synopsis

Description

The Vovida Open Communication and Application Library is an open source SIP proxy, registrar, and feature service platform. It includes a number of servers which are tightly integrated and which can provide SIP call control, routing, and SIP features, such as call forwarding and voice mail.

More information is available from http://www.vovida.org/
