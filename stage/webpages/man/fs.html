fs

Name

fs -- VOCAL feature server

Synopsis

fs [options] [sip-port]

Description

A CPL-based feature server for VOCAL systems. It handles both called and calling features, including call forward on busy, call forward no answer, call blocking. Call forward no answer works with vmserver(1) and ua(1) to provide voice mail features.

Options

-c arg

specify configuration file

-C arg

specify configuration directory

-d

do not fork into background (no daemon mode)

-r

disable SIP retransmission

-f arg

specify log file

-v arg

specify log level

-R arg

initial retransmission time in ms

-M arg

maximum retransmission time in ms

-h

disable multicast heartbeat

-s

use syslog for logging

-n

do not read options from any configuration file

-P arg

default provisioning server

-S arg

backup provisioning server
