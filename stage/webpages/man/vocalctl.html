vocalctl

Name

vocalctl -- control a VOCAL system

Synopsis

vocalctl [-dn] [command]

Description

vocalctl(8) is a frontend to the vocald(8) process controller for VOCAL. It

Commands

vocalctl takes a number of commands, or if no command is passed, it lists the commands which can be used.

respawn <process>

enable automatic respawning of tasks if the task e

config

print vocal.conf configuration values

stop

stop vocald

shutdown

stop vocald (no error if not running)

status

show status of processes

start

start vocald

reload

reread vocal.conf configuration file

norespawn <process>

disable automatic respawning of tasks if the task

disable <process>

disable process (or all if no argument)

enable <process>

enable process (or all if no argument)

restart

restart vocald (or start if not running)

Options

-d

run vocald in debugging mode (and in no daemon mode) to allow debugging messages to appear on the console.

-n

do not interact with the user if there are problems (suitable for scripting).
