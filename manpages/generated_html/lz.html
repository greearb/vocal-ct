lz

Name

lz -- SIP based Load Generator.

Synopsis

lz [options]

Description

SIP Load Generator (ElGee).

ElGee is an implementation on top of the Vovida SIP stack, base code, util and other supporting libraries that genrates SIP calls for load and stress testing. You can test SIP calls originating from ElGee against a SIP proxy server or any other SIP UAS.

This release of ElGee comes with enhanced features:

Authentication

ElGee supports HTTP-Basic, HTTP-Digest and SIP-CHAP based authentication. both WWW-Authenticate and Proxy-Authenticate based challenges can be authenticated for INVITEs and REGISTERs.

Running modes

This release of ElGee supports three running modes :

  • Originating

  • Registration

  • Terminating

These running modes allow for stress load generation to simulate large number of SIP clients.

UDP & TCP

ElGee supports both transport protocols. for originating/registration mode, sample message should specify the transport param in headers, while in terminating mode it listens for both. you can use ElGee more flexibly by using a config file. see the included elgee.conf for different configurable parameters

Options

-f arg

specify configuration file

source user#[@host[:port]]

dest user#@host:port
