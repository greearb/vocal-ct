vocald

Name

vocald -- control a VOCAL system

Synopsis

vocald [-dn] [-f config_file] [-D debug_level]

Description

The vocald server controls the processes in a vocal(8) system. People only wishing to run VOCAL and not debug it should run vocalctl(8) instead.

Options

-d

run vocald in no daemon mode to allow debugging messages to the console.

-D debug_level

set the debugging level.

-f config_file

set the config file (which defaults to /usr/local/vocal/etc/vocal.conf).

See Also

vocalctl(8)vocalconf(5)vocal(8)
