PACKAGES in Vocal 1.5.0
------------------------

Version 1.5.0
Release date: March 2003

1  Vocal
   The package contains set of stacks (SIP, RTP, SDP) and suite of 
   applications SIP Proxy (MS), SIP regsitrar (RS), Provisoning 
   sub-system, voice mail sub-system and feature servers(FS) for 
   call-features. Using an allinone configuration, one can use
   the package to deploy a SIP based VoIP-softswitch.Refere to
   vocal/README_VOCAL.txt for more details.

2. SIPstack
   The package provides SIP stack implementation. The package also
   contains a sample user agent and a back-to-back user agent sample
   implementation. Refer to vocal/README_SIP.txt for more details.

3. RTP
   The package provides RTP stack implementation.Current release
   only tested the support for G711 codec.  Refer to vocal/README_SIP.txt
   for more details.

4. Auto-Provisioning
   Released as part of Vocal, the package provides a framework to 
   enable auto-provisioning of Cisco ATA-186(SIP) into Vocal system.
   Refer to vocal/provisioning/auto/ATA186HowToProvision.txt for 
   more details.
